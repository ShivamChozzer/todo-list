# todo-list
A simple to-do list web app built using HTML, CSS, and JavaScript. It allows users to add tasks with a name and due date, and delete them when completed.

✨ Features
Add tasks with a due date
Delete tasks
Simple and easy-to-use interface

🛠️ Tech Stack
HTML
CSS
JavaScript

🎯 Purpose
This project was built to practice JavaScript basics like DOM manipulation, event handling, and working with user input.
